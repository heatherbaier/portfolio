Projects

Global Education Observatory

I started the Global Education Observatory in 2019 with the goal of building a public database of school-level data on education across the globe. I currently lead a team of undergraduate students part of William & Mary’s GeoLab in developing the database and building machine learning models to fill in gaps in global education data. The website attached is still in development with an initial completed version expected in Winter 2024. The website is built using PHP, the database hosted using AWS EC2 and RDS, and the geospatial data on the maps is rendered using GeoServer. The data behind the project lives at: https://github.com/heatherbaier/geo


GitHub Link: https://github.com/heatherbaier/geo-web

Migration Data Portal

The migration data portal is the prototype of a dashboard designed to predict migration from municipalities in Mexico to the United States. The dashboard was built using Flask and interacts with a PyTorch model trained using satellite imagery and census data from Mexico. Users can toggle the parameters on the left-hand side of the dashboard, then select a group of municipalities on the map and click predict. The map will modify the census parameters based the user changes and output and updated map and predictions.

CheckMySchool Data Portal

I developed the CheckMySchool Data Portal while interning for CheckMySchool in Manila, Philippines in 2018. The dashboard displays school level data for all public schools in the Philippines for multiple years in multiple formats. It also allows for the querying and download of the data in CSV format. The dashboard was designed using RShiny.
